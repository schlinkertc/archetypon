StringTemplate

 StringTemplate (string=None, template:str)

Custom implementation of Pydantic’s Base Model.

Includes _repr_json_ and _repr_html_ methods for nice displays in Jupyter Lab and Jupyter Notebook, respectively.

class IntroduceMe(StringTemplate):
    template:str = "Hi! My name is {first} {last}"
    first: str
    last: str
IntroduceMe(first='Charlie',last='Schlinkert')
stringHi! My name is Charlie Schlinkert
templateHi! My name is {first} {last}
firstCharlie
lastSchlinkert
IntroduceMe.parse_string("Hi! My name is Humphry Bogart")
stringHi! My name is Humphry Bogart
templateHi! My name is {first} {last}
firstHumphry
lastBogart
IntroduceMe("Hi! My name is James Bond")
stringHi! My name is James Bond
templateHi! My name is {first} {last}
firstJames
lastBond
