StringTemplate

 StringTemplate (string=None, template:str)

String Template Model.

Define the class with a template and fields, and it can parse a string that matches the template into the attributes of the model or accept the attributes and create the string.

Useful for path operations and partitions.

class IntroduceMe(StringTemplate):
    template:str = "Hi! My name is {first} {last}"
    first: str
    last: str
me = IntroduceMe(first='Charlie',last='Schlinkert')
assert me.string == 'Hi! My name is Charlie Schlinkert'
display(me)
stringHi! My name is Charlie Schlinkert
templateHi! My name is {first} {last}
firstCharlie
lastSchlinkert
hb = IntroduceMe.parse_string("Hi! My name is Humphry Bogart")
assert hb.first == 'Humphry'
assert hb.last == 'Bogart'
hb

templateHi! My name is {first} {last}
firstHumphry
lastBogart
